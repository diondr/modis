Uraian Materi

Pembelajaran mengenai manajemen user dan group ini dimulai dengan pembahasan mengenai informasi user dan group. Kemudian dilanjutkan file /etc/passwd dan /etc/group. Setelah itu latihan yang berkaitan dengan bahasan tersebut siap Anda kerjakan. Selamat belajar :)

Informasi User dan Group

Ubuntu menyimpan informasi akun dari user dan group pada beberapa file di dalam direktori /etc/. Jika seorang administrator membuat user baru, file tersebut harus diedit atau melalui aplikasi tertentu.

Manajemen user dan group dapat dilakukan melalui aplikasi User Accounts pada Ubuntu dan melalui terminal. Aplikasi User Accounts bersifat Graphical User Interface (GUI) sedangkan terminal berbasis teks. Pada pembelajaran ini manajemen user dan group dilakukan melalui terminal karena mempunayi kelebihan dalam skrip yang mudah diomatisasi dari pada GUI.

File /etc/passwd

File /etc/passwd merupakan file yang berisi daftar user dimana setiap user dipisahkan dengan baris. Adapaun setiap baris berisi informasi mengenai:

  • Username — nama user yang diketik saat login sistem.
  • Password — berisi pasword yang di-enkripsi (ditampilkan x bila shadow password digunakan)
  • User ID (UID) — bilangan numerik yang ekuivalen dengan username yang menjadi acuan sistem.
  • Group ID (GID) — bilangan numerik yang ekuivalen dengan nama group yang menjadi acuan sistem.
  • GECOS — nama histori, kolom ini bersifat opsional dan digunakan untuk menyimpan informasi tambahan (seperti nama lengkap user).
  • Home directory — path absolute (tetap) untuk home directory dari user.
  • Shell — program yang otomatis dijalankan jika user login.

Contoh baris pada file /etc/passwd yaitu:

diondr:x:1000:1000:Dion D. R.,,,:/home/diondr:/bin/bash

Baris di atas menunjukkan user diondr memiliki shadow password, UID dan GID adalah 1000. Home directory pada /home/diondr dan menggunakan shell /bin/bash.

File /etc/group

File /etc/group adalah file yang berisi daftar group dimana setiap group dipisahkan dengan baris. Adapaun setiap baris berisi informasi mengenai:

  • Group name — nama group.
  • Group password — berisi pasword yang di-enkripsi (ditampilkan x bila shadow password digunakan)
  • Group ID (GID) — bilangan numerik yang ekuivalen dengan nama group.
  • Member list — daftar user yang menjadi milik group.

Contoh baris pada file /etc/group yaitu:

adm:x:4:diondr

Baris diatas menunjukkan, group adm menggunakan shadow password, mempunyai GID 4 dan anggota diondr.

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop dengan sistem operasi Ubuntu
Petunjuk
  • Awali setiap aktivitas dengan do'a, semoga mendapat kemudahan dan keberkahan.
  • Baca dan pahami langkah-langkah yang ada dengan seksama.
  • Pastikan komputer dalam keadaan baik dan benar.
  • Jangan meletakkan benda yang dapat mengeluarkan medan elektromagnetik (handphone, magnet, dan sejenisnya) di dekat komputer.
  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Latihan
  1. Hidupkan komputer Anda.
  2. Jalankan terminal dan login sebagai root.
  3. Kerjakan latihan-latihan berikut ini.

LATIHAN 1. MELIHAT FILE /etc/passwd DAN /etc/group

  1. Lihat isi file /etc/passwd dengan perintah:

    $ cat /etc/passwd | more

    Sebutkan kolom apa saja yang terdapat pada setiap baris.

  2. Lihat isi file /etc/group dengan perintah:

    $ cat /etc/group | more

    Sebutkan kolom apa saja yang terdapat pada setiap baris.

LATIHAN 2. MENAMBAH GROUP

  1. Buatlah 3 group baru dengan perintah groupadd. Perhatikan informasi group baru pada file /etc/group.

    $ groupadd teman

    $ groupadd saudara

    $ groupadd tetangga

    $ cat /etc/group

LATIHAN 3. MENAMBAH USER

  1. Buatlah user baru dengan perintah useradd. Perhatikan perubahan isi file /etc/passwd setelah pembuatan user baru. Juga perhatikan apakah home directory setiap user juga dibuat pada saat pembuatan user baru.

    $ useradd silvi

    $ passwd silvi

    $ grep silvi /etc/passwd

    $ useradd -m agus

    $ grep agus /etc/passwd

    $ ls -l /home

  2. Opsi â€“g pada perintah useradd untuk menentukan group dari user yang dibuat.

    $ useradd -g tetangga joko

    $ grep joko /etc/passwd

    $ useradd -mg teman azizah

    $ grep azizah /etc/passwd

    $ ls -l /home

LATIHAN 4. MEMODIFIKASI GROUP DARI USER

  1. Dengan perintah usermod, modifikasi group dari user digunakan untuk memilih suatu group primer dan kemungkinan juga bagian dari group lain (supplementary group).
    Untuk memodifikasi group dari suatu user dapat digunakan perintah usermod.

    $ usermod -g saudara -G teman,tetangga agus

    $ usermod -g teman -G saudara silvi

LATIHAN 5. MELIHAT GROUP DARI USER

  1. Lihat group dari seorang user dengan perintah groups.

    $ groups agus

    $ groups silvi

    $ groups joko

    $ groups azizah

LATIHAN 6. MENGUBAH PASSWORD USER

  1. Root dapat mengubah password dari user.

    $ passwd agus

  2. Password yang diubah dengan perintah usermod merupakan file enkripsi, sehingga tidak dapat digunakan saat login.

    $ useradd -g teman dita

    $ usermod -p dita dita

  3. Cobalah login sebagai dita, apakah bisa?

  4. Cobalah mengubah password user dengan login pada user yang bersangkutan.
    Login sebagai user, dan ubahlah password user.

    $ passwd

LATIHAN 7. MENGHAPUS USER

  1. Hapus user dengan menggunakan perintah userdel. Opsi -r untuk menghapus seluruh isi home directory.

    $ userdel agus

    $ userdel silvi

    $ userdel -r joko

    $ userdel -r azizah

LATIHAN 8. MENGHAPUS GROUP

  1. Hapus group dengan menggunakan perintah groupdel.

    $ groupdel teman

    $ groupdel saudara

    $ groupdel tetangga

LATIHAN 9. MENGHAPUS HOME DIRECTORY

  1. Hapus home directory

    $ rm -fR /home/agus

    $ rm -fR /home/silvi

Rangkuman

Informasi akun dari user dan group disimpan pada beberapa file di dalam direktori /etc/ pada sistem. File /etc/passwd merupakan file yang berisi daftar user dimana setiap user dipisahkan dengan baris. Sedangkan file /etc/group adalah file yang berisi daftar group dimana setiap group juga dipisahkan dengan baris.

  • Perintah menambah user baru yaitu useradd <username>

  • Perintah memodifikasi user yaitu usermod <username>

  • Perintah merubah password dari user yaitu passwd <username>

  • Perintah melihat group dari user yaitu groups <username>

  • Perintah menghapus user yaitu userdel <username>

  • Perintah menambah group baru yaitu groupadd <groupname>

  • Perintah menghapus group yaitu groupdel <groupname>

  • Perintah menghapus home directory yaitu rm -fR /home/<username>
