Tujuan / Uraian Materi / Lembar Kerja / Rangkuman



Tujuan

  • Dapat menjelaskan jenis-jenis kerusakan saat instalasai Ubuntu.
  • Mampu menyajikan data jenis-jenis kerusakan saat instalasai Ubuntu.

Uraian Materi

Pembelajaran tentang jenis kerusakan saat instalasai sistem operasi ini dimulai dengan pembahasan mengenai jenis-jenis kerusakan saat instalasai Ubuntu secara umum. Kemudian dijabarkan satu per satu jenis-jenis kerusakan yang ada. Selamat belajar :)

Jenis Kerusakan

Saat instalasi Ubuntu mungkin Anda mengalami beberapa masalah atau kerusakan yang mengakibatkan proses instalasi terganngu. Ada beberapa jenis kerusakan saat instalasi Ubuntu. Kerusakan yang sering terjadi diantaranya yaitu:

  • File installer (CD/DVD atau ISO) Ubuntu rusak
  • Proses instalasi yang lambat
  • Sistem operasi lainnya terhapus
  • Tidak bisa booting ke Ubuntu (Instalasi secara Virtual)

Mari kita bahas satu per satu dengan memilih menu di bawah ini.

Installer Ubuntu Rusak

Saat instalasi berlangsung mungkin Anda pernah mengalami instalasi terhenti atau error. Bisa jadi file installer, yang berupad CD/DVD atau file ISO Ubuntu yang Anda gunakan rusak atau corrupt. Secara kasat mata memang file tersebut tidak ada bedanya atau "aman-aman" saja, tetapi sebenarnya file Anda rusak.

Untuk itu, ketika Anda selesai mendownload file ISO ubuntu atau mendapatkannya dari teman, sangat disarankan untuk mengecek apakah file tersebut benar-benar valid. Untuk mengecek valid tidaknya file tersebut bisa digunakan metode MD5 (Message-Digest algorithm 5). Dengan MD5 Anda bisa mengecek integritas dan validitas dari file yang Anda miliki apakah sesuai dengan standar dari Ubuntu.

Proses Instalasi Lambat

Kurang telitinya seorang user dalam instalasi Ubuntu bisa mengakibatnya beberapa masalah. Salah satunya yaitu lambatnya proses instalasi. Kesalahan yang terjadi pada tahap awal . User sering kali memberi tanda cek atau mencentang pilihan Download updates while installing atau Install this third-party software.

Jika hal tersebut dilakukan dan komputer Anda terhubung dengan internet, maka proses instalasi akan berjalan lambat. Proses yang lambat itu terjadi karena sistem akan otomatis mendownload paket-paket aplikasi yang diperlukan langsung daris server repository. Anda bisa menghindari masalah tersebut dengan tidak mencentang kedua pilihan di atas.

Sistem Operasi Lain Terhapus

Jika pada komputer Anda sudah ada sistem operasi Windows dan Anda ingin menginstall Ubuntu secara multiboot, hal yang sangat perlu diperhatikan adalah saat pemilihan Installation Type. Ubuntu memberikan pilihan apakah "Erase disk and install ubuntu", "Something else", atau yang lain. Kesalahan terjadi mungkin karena Anda memilih pada opsi "Erase disk and install ubuntu" yang mengakibatkan sistem oprasi sebelumnya menjadi hilang.

Pada modul ini sangat disarankan untuk menggunakan tipe "Something else". Karena dengan tipe tersebut Anda bisa melihat dan mengatur partisi harddisk sebelum ke langkah selanjtunya. Dengan begitu, terhapusnya sistem operasi lain pun dapat dihindari.

Tidak Bisa Booting ke Ubuntu (Virtualisasi)

Saat proses intsalasi Ubuntu secara virtual mungkin Anda tidak bisa booting ke Ubuntu padahal sudah melakukan tahapan sesuai prosedur. Hal tersebut bisa saja terjadi karena processor yang Anda gunakan tidak sesuai dengan kernel yang ada pada installer Ubuntu.

Anda bisa mengatasinya dengan memilih installer Ubuntu yang sesuai dengan tipe processor atau mengatur fitur processor pada VirtualBox atau aplikasi virtualisasi yang Anda gunakan.

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop
  • Buku, artikel, atau sumber lain tentang Linux
Petunjuk
  • Awali setiap aktivitas dengan do'a, semoga mendapat kemudahan dan keberkahan.
  • Baca dan pahami langkah-langkah yang ada dengan seksama.
  • Pastikan komputer dalam keadaan baik dan benar.
  • Jangan meletakkan benda yang dapat mengeluarkan medan elektromagnetik (handphone, magnet, dan sejenisnya) di dekat komputer.
  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Tugas

  1. Pastikan alat dan bahan sudah siap.

  2. Buatlah sebuah tulisan atau artikel tentang jenis-jenis kerusakan saat instalasai Ubuntu, uraikan setiap jenis yang ada.

  3. Buat secara jelas dan semenarik mungkin agar dapat mudah dimengerti baik Anda sendiri maupun orang lain.

  4. Anda bisa menggunakan bantuan komputer atau laptop untuk memperoleh infromasi tambahan.

  5. Kemudian publikasikan tulisan Anda tersebut pada blog, website, atau media sosial.

  6. Jika sudah selesai, jangan lupa matikan komputer/laptop dengan benar.

Rangkuman

Kerusakan saat instalasi Ubuntu mungkin saja terjadi sehingga mengakibatkan proses instalasi terganngu. Ada beberapa jenis kerusakan saat instalasi Ubuntu. Kerusakan yang sering terjadi yaitu:

  • File installer (CD/DVD atau ISO) Ubuntu rusak, diakibatkan karena file tidak valid atau tidak sesuai dengan standar dari Ubuntu.

  • Proses instalasi yang lambat karena user mencentang pilihan Download updates while installing atau Install this third-party software pada tahap awal instalasi.

  • Sistem operasi lainnya terhapus, terjadi karena kesalahan dalam memilih tipe instalasi untuk pengaturan partisi harddisk.

  • Tidak bisa booting ke Ubuntu (Virtualalisasi) dikarenakan processor yang digunakan tidak mendukung kernel yang ada pada installer Ubuntu.
