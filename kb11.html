#11 Instalasi Ubuntu Metode Virtualisasi

Tujuan / Uraian Materi / Lembar Kerja / Rangkuman



Tujuan

Pada pembelajaran ini Anda diharapkan:

  • Dapat menjelaskan konsep instalasi dengan metode virtualisasi.
  • Dapat menjelaskan langkah-langkah instalasi VirtualBox.
  • Mampu membuat komputer virtual baru pada VirtualBox.
  • Mampu melakukan instalasi Ubuntu dengan metode virtualisasi.

Lanjut ke Materi »

Uraian Materi

Pembelajaran mengenai instalasi Ubuntu metode Virtualisasi ini terdiri 2 bahasan utama. Pertama bahasan akan tertuju pada virtualisasi itu sendiri, kemudian dilanjutkan dengan VirtualBox. Selamat belajar :)

Virtualisasi

Sebelum membahas bagaimana menginstal Ubuntu secara virtual, mari kita singgung mengenai apa “virtualisasi” itu sendiri. Secara umum virtualisasi merupakan proses atau cara untuk membuat sesuatu seolah-olah menjadi seperti nyata. Contohnya, kita punya sebuah PC yang di dalamnya terdapat main board, CPU, RAM, harrddisk, dan komponen penyusun lainnya yang dapat dilihat secara fisik. Dengan metode virtualisasi kita bisa “membuat” PC sendiri lengkap dengan komponen-komponennya, hanya saja PC virtual itu tidak dapat dilihat secara nyata komponennya.

Nah, dengan PC atau komputer virtual itu kita bisa melakukan instalasi Ubuntu dengan metode virtualisasi. Maksudnya yaitu kita tidak langsung menginstal Ubuntu pada komputer nyata, melainkan kita “membuat” dahulu komputer secara virtual, baru kemudian kita menginstal Ubuntu pada komputer virtual tersebut.

Instalasi secara virtual ini tidak disarankan untuk penggunaan dalam waktu lama. Kenapa? Karena dengan metode ini komputer asli kita menjadi lebih terbebani dengan 2 sistem operasi yang berjalan sekaligus. Namun, untuk pengenalan ataupun ingin mencoba suatu sistem operasi bisa menggunakan metode virtualisasi ini.

VirtualBox

VirtualBox merupakan suatu aplikasi virtualisasi cross-platform yang dapat dijalankan pada berbagai sistem operasi (Windows, Linux, atau Mac). Dengan aplikasi ini kita dapat membuat komputer virtual baru di dalam komputer nyata yang sedang kita gunakan. Jadi ada beberapa sistem operasi berjalan secara bersamaan pada satu waktu dan pada satu komputer. Contohnya, Anda memiliki komputer yang terinstall sistem operasi Windows, dengan aplikasi VirtualBox Anda bisa menjalankan sistem operasi lain (Ubuntu misalnya) pada komputer tersebut secara bersamaan secara virtual.

Secara umum kita juga bisa mengatur seberapa besar harddisk komputer virtual yang akan dibuat, processor, RAM, dan menambah DVD-drive layaknya merakit komputer asli. Kemudian kita bisa menginstal Ubuntu ke dalam komputer virtual telah dibuat buat.



Lanjut ke Lembar Kerja »

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop (RAM minimal 1 GB)
  • File installer VirtualBox
  • Installer (CD/DVD) Ubuntu 12.04
Petunjuk
  • Awali setiap aktivitas dengan do'a, semoga mendapat kemudahan dan keberkahan.
  • Baca dan pahami langkah-langkah yang ada dengan seksama.
  • Pastikan komputer dalam keadaan baik dan benar.
  • Jangan meletakkan benda yang dapat mengeluarkan medan elektromagnetik (handphone, magnet, dan sejenisnya) di dekat komputer.
  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Latihan

Pada latihan ini Anda akan belajar bagaimana menginstall aplikasi virtualBox dilanjutkan dengan cara membuat komputer baru pada virtualBox. Siap?


Instalasi VirtualBox

  1. Pastikan Anda sudah memiliki file installer VirtualBox, dalam pembelajaran ini digunakan VirtualBox-4.3.8-92456-Win.exe. Anda boleh menggunakan versi yang lain.
  2. Jalankan file tersebut dengan cara double click file tersebut.
  3. Kemudian akan muncul jendela awal instalasi VirtualBox. Klik Next untuk melanjutkan.
  4. Setelah itu muncul pilihan fitur apa saja yang ingin diinstall. Langsung saja klik Next jika Anda ingin fitur standar yang telah ditentukan.
  5. Akan muncul pilihan custom setup untuk pembuatan shortcut dan register file. Klik Next untuk melanjutkan secara default.
  6. Kemudian akan ada peringatan mengenai reset jaringan. Klik Yes untuk melakukan melanjutkan.
  7. Setelah itu muncul konfirmasi untuk memulai instalasi. Klik Install, kemudian proses instalasi akan berjalan.
  8. Setelah selesai, maka akan muncul jendela informasi bahwa instalasi selesai. Centang “Start Oracle Vm VirtualBox” jika ingin langsung menjalankan VirtualBox dan klik Finish.

Membuat Komputer Virtual pada VirtualBox

  1. Buka aplikasi VirtualBox, sehingga muncul jendela awal VirtualBox. Klik New untuk membuat komputer virtual baru.
  2. Kemudian Muncul jendela baru ucapan selamat datang, klik Next untuk melanjutkan.
  3. Beri nama komputer virtual yang dibuat pada kolom Name. Pada OS Type, pilih Linux untuk pilihan Operating System dan untuk Version pilih Ubuntu. Jika sudah, klik Next.
  4. Kemudian muncul pengaturan Memory. Pada percobaan ini diatur RAM sebesar 512 MB, tetapi sesuaikan dengan RAM komputer asli Anda.
    • Jika komputer asli Anda memiliki RAM kurang dari 1 GB maka atur RAM pada VirtualBox sebesar 256 MB.
    • Sedangkan, jika RAM komputer asli Anda lebih dari 1 GB maka atur RAM pada VirtualBox sebesar 512 MB.
    Setelah itu klik Next untuk menuju ke langkah berikutnya.
  5. Pada pengaturan Virtual Hard Disk pilih Create new hard disk. Kemudian untuk melanjutkan klik Next.
  6. Setelah itu pada File type, pilih VDI (VirtualBox Disk Image) lalu klik Next.
  7. Pilih Dynamically allocated pada Storage details, kemudian klik Next untuk melanjutkan.
  8. Kemudian atur Size virtual disk sebesar 8 GB dan untuk melanjutkan klik Next.
  9. Setelah itu muncul Summary sebagai informasi ringkasan virtual disk yang telah Anda buat. Klik Create untuk membuatnya.
  10. Kemudian ada infromasi juga mengenai ringkasan virtual machine atau komputer virtual yang Anda buat. Klik Create untuk memprosesnya.
  11. Jika berhasil maka akan muncul nama komputer virtual yang kita buat pada jedela utama VirtualBox.
Tugas

Setelah instalasi VirtualBox dan pembuatan komputer virtual pada VirtualBox selesai, kini saatnya menginstall Ubuntu pada komputer virtual tersebut. Anda harus “memasukkan” file ISO Ubuntu terlebih dahulu ke dalam komputer virtual. File ISO tersebut sebagai pengganti dari installer CD/DVD Ubuntu pada komputer nyata.

  1. Pertama pilih komputer virtual yang ingin dipakai, dalam percobaan ini yaitu “KomputerKu”, lalu klik Setting.
  2. Kemudian akan muncul jendela baru untuk pengaturan. Pilih Storage. Pada IDE Controller pilih Empty. Setelah itu, pada kolom Attributes, klik ikon CD dan pilih Choose a virtual CD/DVD disk file.
  3. Akan muncul jendela baru untuk memilih file ISO Ubuntu. Arahkan ke dalam folder dimana Anda menyimpan file ISO tersebut. Dalam percobaan ini terletak pada folder Downloads. Jika sudah dipilih, klik Open.
  4. Pastikan pada IDE Controller terdapat nama file ISO yang dimasukkan, dalam hal ini yaitu ubuntu-12.04.4-desktop-i386.iso. Klik OK untuk melanjutkan.
  5. Kini saatnya “menghidupkan” komputer virtual. Tetap dengan komputer virtual yang dipilih (dalam hal ini yaitu KomputerKu), klik Start pada jendela utama Virtualbox.
  6. Setelah itu akan muncul jendela baru komputer virtual dan proses booting Ubuntu sedang berlangsung.

Kini tugas Anda yaitu melanjutkan instalasi Ubuntu secara virtual tersebut. Selamat mengerjakan :)

Rangkuman

Virtualisasi merupakan proses atau cara untuk membuat sesuatu seolah-olah menjadi seperti nyata. PC atau komputer juga bisa dibuat secara virtualisasi. Dengan aplikasi VirtualBox dapat dibuat komputer virtual baru di dalam komputer nyata yang sedang kita gunakan. Secara umum kita juga bisa mengatur seberapa besar harddisk komputer virtual yang akan dibuat, processor, RAM, dan menambah DVD-drive layaknya merakit komputer asli.

Instalasi aplikasi VirtualBox umumnya terdiri dari berapa langkah yaitu (1) perisapan file installer VirtualBox, (2) informasi awal, (3) pilihan fitur, (4) custom setup, (5) reset jaringan, (6) proses instalasi, dan (7) selesai.

Untuk membuat komputer virtual pada VirtualBox ada beberapa langkah yang harus diperhatikan yaitu (1) awal pembuatan komputer virtual, (2) VM Name dan OS Type, (3) pengaturan RAM, (4) pengaturan Virtual Disk, (5) pilihan Virtual Disk, (6) pengaturan detail Virtual Disk, (7) pengaturan ukuran Virtual Disk, (8) ringkasan Virtual Disk, (9) ringkasan Virtual Machine, (10) selesai.

Instalasi Ubuntu dengan metode virtualisasi tidak jauh berbeda dengan instalasi secara langsung pada komputer. Akan tetapi, perlu diatur file ISO Ubuntu pada VirtualBox terlebih dahulu sebagai pengganti dari installer CD/DVD Ubuntu pada komputer asli.
