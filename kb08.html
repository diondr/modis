Partisi Harddisk pada Linux



Harddisk sebagai media penyimpanan pada komputer menjadi bagian yang tidak boleh dipandang sebelah mata. Sistem dan data-data yang ada sebagian besar tersimpan dalam harddsik. Oleh karena itu kita perlu mengenal bagaimana membagi atau mempartisi harddisk dengan baik dan benar. Hal tersebut dilakukan demi keamanan sistem dan data-data tersebut. Jangan sampai data kita yang tersimpan tinggal kenangan akibat salah mempartisi atupun memformat.





JENIS PARTISI PADA LINUX

Pada Linux dan sistem komputer pada umumnya dikenal 3 jenis partisi:

  1. Partisi Primary

    2.

Partisi primary merupakan jenis partisi utama harddisk pada sistem operasi umumnya. Partisi ini hanya bisa dibuat maksimal 4 partisi, hal ini sangat berbeda dengan sistem DOS/Windows yang hanya mengijinkan satu jenis partisi primary untuk sistem . Jenis partisi primary menempati nomor partisi 1, 2, 3 dan 4. Pada Linux misalnya, harddisk sda bisa dibagi menjadi sda1, sda2, sda3, dan sda4.


  1. Partisi Extended

    2.

Partisi extended merupakan partisi perluasan untuk mengatasi kekurangan partisi primary (dimana hanya dimungkinkan adanya 4 partisi). Jika ingin memiliki partisi lebih dari 4 maka partisi extended dibutuhkan. Apakah bisa? Ya, caranya yaitu dengan “mengorbankan ” satu partisi primary kemudian digunakan oleh partisi extended.

Kita tidak dapat menyimpan data pada jenis partisi ini. Di dalam partisi extended ini harus dibuat partisi logical terlebih dahulu agar dapat digunakan untuk menyimpan data. Jumlah partisinya pun bisa lebih banyak.

Partisi extended selalu menempati nomor partisi +1 dari partisi primary. Misalnya ada 2 jenis partisi primary pada harddisk (sda1 dan sda2), maka posisi partisi extended adalah sda3.


  1. Partisi Logical

    2.

Jenis partisi ini selalu dibuat di dalam partisi extended. Nomor partisi selalu dimulai dari 5 sampai seterusnya. Misalnya ada 3 jenis partisi logical pada suatu harddisk sda, maka masing-masing partisi logical akan menempati sda5, sda6, dan sda7.

Untuk mempermudah pemahaman mari kita lihat contoh berikut. Suatu komputer memiliki harddisk sebesar 320 GB (sda). Harddisk bisa dibagi menjadi 50 GB (sda1), 50 GB (sda2), 100 GB (sda3), dan 120 GB (sda4). Pada sda4 akan dijadikan partisi extended. Kemudian diisi dengan partisi logical sebanyak 3 partisi. Besarnya masing-masing 60 GB (sda5), 40 GB (sda6), dan 30 GB (sda7) seperti ditunjunkkan pada gambar 8.2.

Sistem File

Semua informasi yang tersimpan dalam media penyimpanan (harddisk, flash drive, CD, dll) merupakan sebuah file. Pada Linux, seluruh program, dokumen, konfigurasi sistem, semuanya tersimpan dalam ribuan file. Semua file tersebut terorganisasi dalam sebuah sistem file.


ATURAN PENAMAAN FILE PADA LINUX

Penamaan file, links, dan direktori (folder) pada Linux mengikuti aturan sebagai berikut:

  • Maksimal nama file adalah 256 karakter, sedangkan nama direktori maksimal 4096 karakter.
  • Case sensistive, dibedakan antara huruf besar dan kecil. Misal, "TKJ" berbeda dengan "Tkj".
  • Dapat menggunakan huruf maupun angka.
  • Untuk menghidari kesalahan penulisan, sebaiknya tidak menggunakan karakter "#". Karena kalimat setelahnya akan dianggap sebagai komentar.

TIPE FILE

Pada umumnya, Linux memiliki beberapa tipe file yang digunakan yaitu:

  • Executable file, file-file yang berisi instruksi program untuk dijalankan oleh sistem.
  • System data file, file-file ini berisi informasi yang digunakan oleh program atau script. Biasanya digunakan oleh administrator atau para programmer untuk menjalankan program sesuai kebutuhan.
  • User data file, file-file ini berisi teks dan data yang dibuat oleh user.

Secara default Linux dapat mengenali secara otomatis seluruh tipe file sehingga Anda tidak perlu bingung program apa yang digunakan untuk membuka file-file tersebut.


EKSTENSI FILE LINUX

Ekstensi file yang digunakan Linux sangat beragam. Namun, beberapa ekstensi file yang sering ditemuai adalah sebagai berikut:

.gz File kompresi dengan menggunakan kompresi gzip.

.tar File yang berisi bundelan/kumpulan file lain yang dijadikan satu.

.tar.gz File bundelan yang terkompresi dalam format .gz.

.deb Paket instalasi debian GNU/Linux.

.bz2 File kompresi dengan menggunakan kompresi bzip2.

.c File yang ditulis dalam bahasa C.

.conf File konfigurasi GNU/Linux.

.so Objek share (pustaka).

.src File source code. Yang ditulis dalam plain text, file tersebut harus dikompilasi terlebih dahulu sebelum menggunakannya.

Selain beberapa eksensi file tersebut, Linux juga dapat membaca ekstensi file dari sistem operasi lain seperti *.jpg untuk file gambar, *.html untuk file web, dan *.doc untuk word processor Microsoft . Bahkan Linux dapat membaca file *.exe dari Windows menggunakan program emulator seperti Wine.

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop
  • Installer (CD/DVD) Ubuntu 12.04
Petunjuk

  • Pastikan komputer dalam keadaan baik dan benar.

  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Latihan

Pada latihan ini Anda akan berkenalan dengan GParted, sebuah aplikasi untuk mengatur partisi harddisk.

  1. Hidupkan komputer dan masukkan CD/DVD installer Ubuntu 12.04 ke dalam komputer.
  2. Atur BIOS agar booting pertama melalui CD/DVD.
  3. Jika sudah maka akan tampil proses booting Ubuntu seperti gambar 8.4.
  4. Kemudian akan muncul tampilan baru dengan 2 opsi, yaitu “Try Ubuntu” dan “Install Ubuntu”.
    • “Try Ubuntu”, Anda hanya mencoba Ubuntu secara Live-CD tanpa menginstallnya ke dalam komputer.
    • Sedangkan “Install Ubuntu”, Anda langsung menginstal Ubuntu ke dalam komputer.

    Pada latihan ini, pilih Try Ubuntu.

  5. Jika sudah muncul desktop awal Ubuntu, klik menu Dash Home (ikon Ubuntu kiri atas). Lalu ketikan “gparted” (tanpa tanda petik) dan klik ikon GParted untuk menjalankannya.

  6. Aplikasi GParted akan berjalan dan akan terlihat susuan partisi harddisk (gambar 8.7). Susunan ini bisa jadi berbeda dengan susunan partisi harddisk Anda.

  7. Untuk melihat informasi suatu partisi, klik kanan pada partisi yang diinginkan, pilih Information. Maka akan tampil informasi partisi tersebut seperti pada gambar 8.8.

  8. Anda bisa juga melihat harddisk yang Anda gunakan mendukung file system apa saja. Pilih View pada menu bar, kemudian klik File System Support. Maka akan tampil jendela File System Support yang akan menampilkan infromasi support harddisk yang Anda gunakan.

  9. Melalui GParted Anda juga bisa memformat suatu partisi. Klik kanan pada partisi yang diinginkan, pilih Format to. Kemudian Anda bisa memilih jenis file system yang diinginkan.

Tugas

  1. Pada latihan sebelumnya Anda telah berkenalan dengan GParted. Pada susunan partisi harddisk, ada 6 partisi pada harddisk tersebut.
    Jelaskan manakah yang termasuk Primary, extended, & logical parition pada susunan partisi tersebut!

  2. Pada gambar 8.11 di atas juga dapat dilihat bahwa File System yang digunakan yaitu ext4, fat32, extented, linux-swap, & ext2. Jelaskan perbedaan masing-masing File System tersebut!

Rangkuman

Linux dan sistem komputer pada umumnya mengenal 3 jenis partisi, yaitu:

  • Partisi Primary, merupakan jenis partisi utama harddisk pada sistem operasi.
  • Partisi Extended yaitu partisi perluasan untuk mengatasi kekurangan partisi primary.
  • Partisi Logical merupakan partisi yang dibuat di dalam partisi extended.

File merupakan semua informasi yang tersimpan dalam media penyimpanan (harddisk, flash drive, CD, dll). Linux pada umumnya memiliki 3 tipe file yang digunakan yaitu executable file, system data file, dan user data file. Ekstensi file yang digunakan Linux sangat beragam, diantaranya yaitu .gz, .tar, .tar.gz, .deb, .bz2, .c, .conf, .so, dan .src.
