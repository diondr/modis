#9 Instalasi Ubuntu metode Clean Install & Upgrade

Tujuan / Uraian Materi / Lembar Kerja / Rangkuman



Tujuan

Pada pembelajaran ini Anda diharapkan:

  • Dapat menjelaskan langkah-langkah instalasi Ubuntu dengan metode clean install.
  • Dapat menjelaskan langkah-langkah instalasi Ubuntu dengan metode sistem upgrade.
  • Mampu melakukan instalasi Ubuntu dengan metode clean install.
  • Mampu melakukan instalasi Ubuntu dengan metode sistem upgrade.

Lanjut ke Materi »

Uraian Materi

Pembelajaran mengenai instalasi Ubuntu metode clean install & upgrade ini terdiri 2 bahasan utama. Pertama bahasan akan tertuju pada instalasi Ubuntu dengan metode clean install, kemudian dilanjutkan dengan instalasi Ubuntu dengan metode sistem upgrade. Selamat belajar :)

Metode Instalasi Clean Install

Salah satu metode atau cara untuk menginstall Ubuntu adalah dengan metode clean install. Dengan cara ini kita akan mengistall Ubuntu secara langsung pada komputer atau laptop secara fresh. Maksudnya komputer kita belum ada sistem operasi yang terinstall atau sudah ada dan ingin diganti dengan Ubuntu. Nantinya hanya siatem operasi Ubuntu yang terinstall pada komputer.

Secara umum ada 10 langkah untuk mengistall Ubuntu dengan clean install, yaitu:

  1. Booting installer Ubuntu (melalui CD/DVD atau media lainnya)
  2. Pemilihan bahasa
  3. Informasi persiapan instalasi
  4. Pilihan tipe instalasi
  5. Konfirmasi penghapusan harddisk
  6. Pengaturan lokasi
  7. Pengaturan keyboard layout
  8. Pengisian data diri
  9. Proses instalasi Ubuntu
  10. Selesai (restart)
Metode Instalasi Sistem Upgrade

Metode ini dilkukan jika komputer Anda sudah terinstall sistem operasi Ubuntu, kemudian Anda ingin mengupgrade atau memperbarui ke versi yang lebih baru. Contohnya pada komputer sudah terinstall Ubuntu 12.04 dan akan diupgrade ke versi terbaru Ubuntu 13.10. Jadi Anda perlu menyiapkan installer Ubuntu dengan versi yang lebih baru dari pada yang terinstall pada komputer.

Langkah-langkah untuk menginstall Ubuntu dengan metode sistem upgrade tidak jauh berbeda dengan metode clean install. Hanya saja pada langkah “pilihan tipe instalasi” Anda perlu mengaturnya pada pilihan “upgrade”. Dengan metode ini, Anda akan mendapatkan Ubuntu dengan versi yang lebih baru terinstall pada komputer Anda.



Lanjut ke Lembar Kerja »

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop
  • Installer (CD/DVD) Ubuntu 12.04
  • Installer (CD/DVD) Ubuntu 13.10
Petunjuk
  • Awali setiap aktivitas dengan do'a, semoga mendapat kemudahan dan keberkahan.
  • Baca dan pahami langkah-langkah yang ada dengan seksama.
  • Pastikan komputer dalam keadaan baik dan benar.
  • Jangan meletakkan benda yang dapat mengeluarkan medan elektromagnetik (handphone, magnet, dan sejenisnya) di dekat komputer.
  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Latihan

Pada latihan ini Anda akan belajar bagaimana menginstall Ubuntu dengan metode clean install. Siap?

  1. Hidupkan komputer dan masukkan CD/DVD installer Ubuntu 12.04 ke dalam komputer yang akan diinstall.
  2. Atur BIOS agar booting pertama melalui CD/DVD.
  3. Jika sudah maka akan tampil proses booting Ubuntu.
  4. Kemudian akan muncul tampilan baru dengan 2 opsi, yaitu “Try Ubuntu” dan “Install Ubuntu”.
    • “Try Ubuntu”, Anda hanya mencoba Ubuntu secara Live-CD tanpa menginstallnya ke dalam komputer.
    • Sedangkan “Install Ubuntu”, Anda langsung menginstal Ubuntu ke dalam komputer.
    Untuk saat ini kita langsung menginstall Ubuntu, jadi pilih Install Ubuntu.
  5. Setelah itu akan tampil informasi persiapan instal Ubuntu. Klik Continue untuk melanjutkan.
  6. Lalu akan diminta pemilihan pengaturan disk sebagai tempat instalasi Ubuntu. Karena metode yang kita gunakan saat ini yaitu clean install maka pilih saja Erase disk and install Ubuntu. Lalu klik Continue.
  7. Akan muncul konfirmasi untuk mengahpus disk dan menginstal Ubuntu. Klik Install Now untuk menginstal Ubuntu.
  8. Proses instalasi mulai berjalan, Anda diharuskan mengisi data-data yang dibutuhkan. Pertama, Anda diminta untuk mengisi lokasi tempat tinggal. Dalam latihan ini Anda bisa memilih Jakarta dengan mengklik posisi yang sesuai atau mengetikkan pada kolom yang disediakan. Jika sudah, klik Continue.
  9. Seletah itu pilih English (US) untuk pilihan keyboard layout lalu klik Continue.
  10. Kemudian Anda diminta untuk mengisi data diri. Silakan isi nama, nama komputer, username, password dan pada pilihan login pilih Require my password to log in. Terakhir klik Continue.
  11. Proses instalasi Ubuntu secara clean install sedang berjalan. Sambil menunggu proses instalasi, Anda disuguhi tampilan slide sekilas mengenai Ubuntu.
  12. Setelah proses instalasi selesai, muncul peringatan Installation Complete. Klik Restart Now.
  13. Kemudian muncul peringatan untuk mengeluarkan media instalasi, keluarkan CD/DVD dari komputer Anda lalu tekan Enter pada keyboard untuk memrposesnya.
  14. Setelah itu komputer akan melakukan restart dan Ubuntu siap untuk digunakan.
Tugas

Setelah instalasi Ubuntu dengan metode clean install selesai, sekarang Anda telah memiliki sistem operasi Ubuntu terpasang pada komputer. Anda bisa melihat versi Ubuntu yang terinstall dengan cara search "Details" pada desktop awal Ubuntu (gambar 9.14). Hasilnya terlihat seperti pada gambar 9.15.

Tugas Anda sekarang yaitu melakukan upgrade Ubuntu ke versi yang lebih baru melalui installer CD/DVD. Hasilnya seperti gambar 9.16 di bawah ini.

Setelah selesai, Anda bisa menuliskan atau menceritakan apa saja yang Anda peroleh pada pembelajaran kali ini. Kemudian publikasikan tulisan Anda tersebut pada blog, website, atau media sosial. Selamat mengerjakan :)

Rangkuman

Melalui metode instalasi clean install kita dapat mengistall Ubuntu secara langsung pada komputer dengan sistem yang baru. Secara umum ada 10 langkah untuk mengistall Ubuntu dengan clean install, yaitu (1) booting installer Ubuntu (melalui CD/DVD atau media lainnya), (2) pemilihan bahasa, (3) informasi persiapan instalasi, (4) Pilihan tipe instalasi, (5) konfirmasi penghapusan harddisk, (6) pengaturan lokasi, (7) pengaturan keyboard layout, (8) pengisian data diri, (9) proses instalasi Ubuntu, dan (10), selesai (restart).

Metode selanjutnya yaitu metode instalasi sistem upgrade, metode ini dilkukan untuk memperbarui Ubuntu ke versi yang lebih baru. Langkah-langkah untuk menginstall Ubuntu dengan metode sistem upgrade tidak jauh berbeda dengan metode clean install. Hanya saja pada langkah “pilihan tipe instalasi” Anda perlu mengaturnya pada pilihan “upgrade”.
