Uraian Materi

Pembelajaran ke-2 ini Anda akan mempelajari arsitektur sistem operasi open source Linux. Arsitektur sistem operasi Linux terdiri dari 4 komponen utama yaitu kernel, shell, utilitas, dan aplikasi. Mari kita bahas satu per satu komponen tersebut.

Kernel

Kernel merupakan inti dari sistem Linux yang mengontrol perangkat keras dan membentuk fungsi dasar. Fungsi-fungsi tersebut antara lain:

  • Pelayanan tanggal dan jam sistem
  • Manajemen berkas dan penanganan sekuriti
  • Pelayanan operasional input output (I/O)
  • Manajemen dan penjadwalan proses
  • Manajemen memori
  • Melakukan kegiatan akuntansi sistem
  • Melakukan penanganan kesalahan dan iterupsi

Bagian lain dari Linux, seperti aplikasi dan shell, memanggil kernel untuk membentuk sebuah layanan. Linux menyediakan anatarmuka antara bagian luar kernel dan kernel sendiri berupa system call (pemanggilan sistem). Setiap pemanggilan sistem memerintahkan kernel untuk membentuk layanan tertentu sesuai permintaan program yang memanggilnya. Untuk lebih jelasnya dapat diilustrasikan pada gambar di bawah ini.

Shell

Shell adalah penerjemah (command line interpreter) pada sistem Linux. Sering kali shell disebut juga sebagai terminal. Perangkat lunak inilah yang menjadi jembatan antara user dengan sistem Linux. User cukup memberikan perintah kemudian shell akan menangani­nya. Perintah tersebut dapat berupa:


  • Perintah built-in yaitu perintah yang merupakan bagian internal dari shell .

  • Perintah eksternal , perintah yang bukan bagian internal dari shell (dapat berupa utilitas atau program aplikasi) .

Baik pada Linux maupun UNIX, shell digunakan untuk interaksi antara user dengan komputernya termasuk juga dalam mengontrol session UNIX dan pemrograman. Unix shell menyediakan sekumpulan instruksi khusus yang dapat diguna­kan untuk membuat program shell scripts. Ada beberapa jenis shell yang umum ditemukan dalam distro Linux, yaitu Bash dan tcsh.

Utilitas

Utilitas (utility) merupakan program yang disediakan sistem Linux untuk melaksanakan tugas tertentu. Banyak sekali utilitas dengan berbagai fungsinya. Beberapa kelompok utilitas dilihat dari fungsinya yaitu:

  • Utilitas manajemen berkas dan direktori
    Utilitas kelompok ini sangat bemanfaat untuk melakukan tugas yang berhubungan dengan berkas dan direktori, misalnya untuk membuat direktori dan menghapus berkas.

  • Utilitas penyunting berkas
    Utilitas ini sering disebut editor, sangat bermanfaat untuk membuat program atau menyimpan informasi tertentu ke dalam berkas.

  • Utilitas penunjang komunikasi dan jaringan
    Utilitas ini bermanfaat untuk melakukan komunikasi antar pemakai. Bahkan pemakai dapat mentransfer data antar sistem.

  • Utilitas administrasi sistem
    Utilitas ini berguna bagi administrator sistem untuk mengelola sistem. Misalnya untuk menciptakan nama pemakai baru dan mendaftarkan printer pada sistem.

  • Utilitas penganalisis unjuk kerja sistem
    Utilitas ini dapat digunakan oleh administrator sistem untuk menganalisis unjuk kerja sistem, sehingga dapat melakukan pengaturan guna meningkatkan unjuk kerja.

  • Utilitas pemrograman C.
    Utilitas ini berguna untuk membuat program aplikasi dengan bahasa pemrograman C

  • Utilitas untuk keperluan backup dan restore
    Utilitas ini bermanfaat untuk menyalin atau memindahkan data atau program ke media eksternal (CD, flashdisk, dll) ataupun sebaliknya, yaitu dari media eksternal ke komputer.

Aplikasi

Aplikasi merupakan program-program yang dibuat oleh user untuk memenuhi kebutuhannya dalam menggunakan komputer. Program-program tersebut dapat dibuat dengan menggunakan sejumlah utilitas, perintah built-in pada shell, atau dibuat dengan bahasa pemrograman seperti C, COBOL, atau Python dan berbagai development tool seperti Oracle dan Informix.

Aplikasi juga dapat berupa suatu paket yang diperoleh dari pemasok perangkat lunak. Pada distro Linux, ubuntu misalnya, banyak menyediakan aplikasi yang siap digunakan seperti aplikasi perkantoran LibreOffice, browser Mozilla Firefox, editor gambar GIMP, editor suara Audacity dan masih banyak lagi yang dapat kita install untuk menggunakannya lebih lanjut.

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop
  • Buku, artikel, atau sumber lain tentang Linux
Petunjuk
  • Awali setiap aktivitas dengan do'a, semoga mendapat kemudahan dan keberkahan.
  • Baca dan pahami langkah-langkah yang ada dengan seksama.
  • Pastikan komputer dalam keadaan baik dan benar.
  • Jangan meletakkan benda yang dapat mengeluarkan medan elektromagnetik (handphone, magnet, dan sejenisnya) di dekat komputer.
  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Tugas

  1. Pastikan alat dan bahan sudah siap.

  2. Buatlah sebuah tulisan atau artikel mengenai arsitektur sistem operasi Linux, termasuk kernel, shell, ulititas, dan aplikasi. Buat semenarik mungkin agar dapat mudah dimengerti baik Anda sendiri maupun orang lain.

  3. Anda bisa menggunakan bantuan komputer atau laptop untuk memperoleh infromasi tambahan.

  4. Kemudian publikasikan tulisan Anda tersebut pada blog, website, atau media sosial.

  5. Jika sudah selesai, jangan lupa matikan komputer/laptop dengan benar.
