Konsep Manajemen Aplikasi

Sistem Linux memiliki manajemen paket aplikasi yang berbeda sesuai dengan distribusi atau distronya. Aplikasi di dalamnya dapat ditambah atau dikurangi dengan beberapa cara. Manajemen aplikasi yang paling populer adalah Debian Package (dpkg) pada distro debian dan Red Hat Package Management (RPM) pada distribusi berbasis Red Hat.

Ubuntu sebagai turunan dari Debian, maka dalam sistemnya pun mengadopsi milik Debian. Dalam pembelajaran ini akan dibahas manajemen aplikasi melalui dpkg dan apt.

Manajemen Aplikasi dengan dpkg

dpkg merupakan suatu sistem untuk memanajemen paket Debian. dpkg juga digunakan pada distro varian debian lainnya, termasuk Ubuntu. File yang digunakan dpkg adalah yang berjenis .deb

Jika menggunakan dpkg kita tidak harus terkoneksi dengan internet. Asalkan tersedia file .deb pada komputer maka file tersbut bisa langsung diintsall pada komputer. Namun, jika ada paket lain yang dibutuhkan (dependency) kurang, maka kita harus mendownload atau mengisntall paket lain tersebut terlebih dahulu.

Manajemen Aplikasi dengan apt

apt merupakan perintah yang dijalankan melalui terminal untuk mencari dan memanajemen paket aplikasi. Manajemen melalui apt terbilang lebih "canggih" daripada dpkg karena bisa menginstall paket aplikasi beserta file-file lain yang dibutuhkan langsung dari server repository. Akan tetapi, karena langsung dari server repository maka dibutuhkan koneksi internet untuk menjalankan apt ini.

Perintah yang umum digunakan dalam apt yaitu apt-get dan apt-cache. apt-get digunakan untuk mendownload dan menginstall paket aplikasi, sedangkan apt-cache untuk mencari informasi tentang paket aplikasi.

Lembar Kerja

Alat dan Bahan
  • Komputer atau laptop dengan sistem operasi Ubuntu
  • Koneksi internet
  • File ubuntu-tweak_0.8.6-1_all.deb, bisa Anda download di http://ubuntu-tweak.com/
Petunjuk
  • Awali setiap aktivitas dengan do'a, semoga mendapat kemudahan dan keberkahan.
  • Baca dan pahami langkah-langkah yang ada dengan seksama.
  • Pastikan komputer dalam keadaan baik dan benar.
  • Jangan meletakkan benda yang dapat mengeluarkan medan elektromagnetik (handphone, magnet, dan sejenisnya) di dekat komputer.
  • Gunakan komputer sesuai fungsinya dengan hati-hati.
  • Kerjakan secara baik, benar, dan sabar.
Latihan
  1. Hidupkan komputer Anda.
  2. Jalankan terminal dan login sebagai root.
  3. Kerjakan latihan-latihan berikut ini.

LATIHAN 1. MANAJEMEN APLIKASI DENGAN dpkg

  1. Menginstall file *.deb dengan perintah:

    $ dpkg -i <namaFile>

    Contoh akan menginstall aplikasi "Ubuntu Tweak" dimana sudah ada file ubuntu-tweak_0.8.6-1_all.deb yang disiapkan.

    $ dpkg -i ubuntu-tweak_0.8.6-1_all.deb

  2. Mengetahui apakah paket aplikasi terinstall atau tidak:

    $ dpkg -s <namaPaket>

    Contoh:

    $ dpkg -s gedit

    $ dpkg -s audacious

    $ dpkg -s firefox | more

  3. Menghapus paket aplikasi dengan perintah:

    $ dpkg -r <namaPaket>

    Contoh akan menghapus aplikasi "Ubuntu Tweak". Tidak perlu menulis nama file ubuntu-tweak_0.8.6-1_all.deb secara penuh, cukup nama aplikasinya saja.

    $ dpkg -r ubuntu-tweak



LATIHAN 2. MANAJEMEN APLIKASI DENGAN apt

  1. Pastikan komputer Anda terhubung dengan internet untuk mejalankan latihan ini.
  2. Lakukan update terlebih dahulu dengan perintah:

    $ apt-get update

    Tunggu sampai proses update selesai.

  3. Menginstall suatu paket aplikasi:

    $ apt-get install <namaPaket>

    Contoh akan menginstall game "snake4":

    $ apt-get install snake4

  4. Mengetahui info suatu paket aplikasi:

    $ apt-cache show <namaPaket>

    Contoh:

    $ apt-cache show snake4

    $ apt-cache show audacious | more

  5. Memperbarui paket aplikasi ke versi yang lebih baru:

    $ apt-get update

    $ apt-get upgrade

  6. Menghapus paket aplikasi:

    $ apt-get remove <namaPaket>

    Contoh akan menghapus game "snake4":

    $ apt-get remove snake4

  7. Menghapus paket aplikasi besera file konfigurasi:

    $ apt-get purge <namaPaket>

    Contoh: $ apt-get purge snake4


Tugas
  1. Cari informasi tentang aplikasi "geany" apakah sudah terinstall atau belum.
  2. Jika belum, install aplikasi tersebut dengan apt.
  3. Setelah selesai, coba jalankan aplikasi tersebut.
  4. Hapus aplikasi tersebut beserta file konfigurasinya.
  5. Cari informasi aplikasi tersebut apakah sudah terhapus atau belum.

  6. Analisa hasil dari tugas yang Anda lakukan.

  7. Jika sudah selesai, Anda bisa menuliskan atau menceritakan apa saja yang Anda peroleh pada pembelajaran kali ini. Kemudian publikasikan tulisan Anda tersebut pada blog, website, atau media sosial.
